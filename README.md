## Installation

```bash
git clone https://github.com/developerkhaled1/choose-lucky-one.git
```

```bash
cd choose-lucky-one
```

```bash
npm i
```

```bash
code .
```

```bash
npm start
```

## Prerequisites

The following open source packages are used in this project:
* React JS v18
* Raw CSS
* React Icons
* Sweet Alert 2
